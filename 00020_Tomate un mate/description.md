Cuando trabajamos con variables, actualizaciones como duplicar, triplicar, incrementar en uno o en una cierta cantidad son tan comunes que JavaScript presenta algunos atajos:

```javascript
x += y // equivalente a x = x + y
x *= y // equivalente a x = x * y
x -= y // equivalente a x = x - y
```
